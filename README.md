# 🐍 贪吃蛇游戏 - 小爪特别版

一个专为Discord环境设计的文字版贪吃蛇游戏，支持状态保持和连续游戏。

## 🎮 游戏特色

- **纯文本界面** - 完美适配Discord环境
- **状态保持** - 支持连续游戏，不会丢失进度
- **多种控制方式** - 支持中文方向和WASD键
- **精美界面** - 使用表情符号的视觉效果
- **完整游戏逻辑** - 碰撞检测、得分系统、游戏结束判定

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/zccmman/snake-game-claw.git
cd snake-game-claw
```

### 2. 运行游戏
```bash
python3 persistent_snake_game.py
```

### 3. Discord中使用
在Discord中输入命令：
- `开始游戏` - 开始新游戏
- `上/下/左/右` - 控制移动
- `重新开始` - 重置游戏

## 🎯 游戏控制

**方向控制：**
- `上` 或 `w` 或 `↑` - 向上移动
- `下` 或 `s` 或 `↓` - 向下移动  
- `左` 或 `a` 或 `←` - 向左移动
- `右` 或 `d` 或 `→` - 向右移动

**其他命令：**
- `开始游戏` - 开始新游戏
- `重新开始` - 重置游戏
- `状态` - 查看当前游戏状态
- `帮助` - 显示帮助信息

- **分数系统**: 每吃一个食物得10分
